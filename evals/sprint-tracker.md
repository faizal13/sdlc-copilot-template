# Sprint-Over-Sprint Agent Performance Tracker

Track agent quality across sprints. Fill in after running `@eval-runner` or manual evaluation on 3-5 representative stories per sprint.

---

## How to Use

1. At the end of each sprint, pick 3-5 stories (mix of simple, medium, complex)
2. For each story, score the agent output using `evals/scoring-rubric.md`
3. Record per-agent averages below
4. Compare trends — are agents improving?
